Installation

Requisitos

  • Android Studio Arctic Fox o superior
  • Emulador o dispositivo físico
  • Conexión al backend en Strapi (configurado en local o remoto)

Pasos para compilar

  1. Clona el repositorio: git clone https://github.com/pablitocavaz04/TrainingRoutine
  2. Abre el proyecto en Android Studio
  3. Configura los ficheros secrets.properties con la URL del backend
  4. Ejecuta el proyecto en un emulador o dispositivo
