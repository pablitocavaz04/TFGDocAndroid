Api

El backend está construido en Strapi. Estas son las entidades principales que consume la app:

  • Usuarios: Registro, login, rol de jugador o entrenador
  • Entrenamientos: tipo, descripción, fecha
  • Sesiones: jugadores, entrenamientos, estado, dirección, imagen

Las peticiones se realizan usando Retrofit con endpoints como:

GET /api/entrenamientos
POST /api/sesiones
POST /api/auth/local
